# Continual Learning

Use this skill when you want to capture lessons learned and feed them back into future execution quality.

## Focus Areas
- Post-task and post-incident retrospectives
- Root-cause to prevention-rule translation
- Updating instructions, checklists, and memory artifacts
- Continuous quality improvement over repeated cycles

## Usage Triggers
- "continual learning"
- "lessons learned"
- "retrospective"
- "prevent this again"
- "improve the process"
