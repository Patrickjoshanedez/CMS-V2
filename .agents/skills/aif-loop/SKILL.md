# AIF Loop

Use this skill when iterating on a solution through repeated analyze, implement, and feedback cycles.

## Focus Areas
- Tight analyze-build-verify loops
- Fast feedback integration from tests and runtime output
- Small safe increments over big risky changes
- Explicit completion criteria at each loop

## Usage Triggers
- "iterate"
- "feedback loop"
- "refine this"
- "stabilize"
- "optimize with evidence"
