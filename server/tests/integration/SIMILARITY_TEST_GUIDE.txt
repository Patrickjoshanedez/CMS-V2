🔧 KEY IMPLEMENTATION DETAILS

Similarity Algorithm:
  - Jaccard Similarity (set intersection/union)

Score Formula:
  score = (|Set A ∩ Set B|) / (|Set A ∪ Set B|)
  overall = Σ(field_scores) / number_of_fields
